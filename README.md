# Souf — دليلك الرقمي لمدينة الوادي

![Souf Logo](https://img.shields.io/badge/Souf-El%20Oued%20Digital%20Guide-orange)

**Souf** هو الدليل الرقمي الشامل لمدينة الوادي في الجزائر. منصة تجمع أفضل المحلات، المطاعم، والخدمات في مكان واحد.

## 🚀 المميزات

- 🔍 بحث فوري مع نتائج فورية
- 📂 7 تصنيفات رئيسية (أزياء، إلكترونيات، سوبرماركت، عطور، أثاث، مطاعم، خدمات)
- ⭐ نظام تقييمات ومراجعات
- 📱 تواصل مباشر عبر واتساب
- 🗺️ خرائط Google مدمجة
- 🌙 وضع داكن / فاتح
- 📱 تصميم متجاوب بالكامل
- ⚡ لوحة تحكم إدارية

## 📁 هيكل المشروع

```
souf-project/
├── index.html          # الصفحة الرئيسية
├── css/
│   └── style.css       # الأنماط
├── js/
│   └── app.js          # المنطق والبيانات
└── assets/             # الصور والموارد
```

## 🛠️ التقنيات المستخدمة

- HTML5 Semantic
- CSS3 (Variables, Grid, Flexbox, Animations)
- Vanilla JavaScript (ES6+)
- GSAP (Animations)
- Font Awesome (Icons)
- Google Fonts (Noto Sans Arabic, Inter, Space Grotesk)

## 🎯 الرؤية

> أن نصبح البوابة الرقمية الافتراضية لجميع المدن الجزائرية، بدءاً من الوادي.
