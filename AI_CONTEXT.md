# Souf AI Context

## Project Name
Souf

## Vision
Souf is a digital platform focused on El Oued Province, Algeria.
The goal is to become the main digital gateway for businesses, services, and local life.

## Current Scope
- Business directory
- Store pages
- Categories
- Search
- Favorites
- Contact
- Admin panel

## Target Users
- Citizens of El Oued
- Store owners
- Visitors

## Design Rules
- Modern
- Fast
- Mobile First
- Arabic RTL
- Gold + White + Black identity
- Simple UI
- High performance

## Technologies
- HTML
- CSS
- JavaScript
- GitHub
- Vercel
- Supabase

## AI Development Rules

- Never create a new project.
- Always improve the existing project.
- Keep file names unless necessary.
- Do not break existing features.
- Keep code clean.
- Keep components reusable.
- Optimize performance.
- Mobile first.
- Arabic RTL by default.

## Long Term Goal

Become the largest local platform for El Oued before expanding to all Algeria.
